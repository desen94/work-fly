## Структура проекта
* Каждая компонента должна иметь свою папку
* Весь набор технологий (README, tsx, styled-components, unit-тесты и т.д.) реализуется в той же папке компоненты, в которой она находится.-->
* В каждой компоненте обязательно должен быть readme файл со скриншотом компоненты, кратким описанием ее работы и с передаваемыми пропсами
```
├── components/              — Папка с реакт компонентами
|   ├── Component/           — Папка реакт компоненты
    |   ├── Component.constants/ — Папка с константами для компоненты
    |   |    ├── index.ts        — Файл с константами
    |   ├── Component.i18n/      — Папка с переводами
    |   |    ├── ru.ts           — Перевод на русский
    |   |    ├── en.ts           — Перевод на английский
    |   |    └── ...
    |   ├── Component.hooks/     — Папка с хуками для компоненты
    |   |    ├── useUpdate.ts    — Файл с хуком для компоненты
    |   |    ├── useNoop.ts      — Файл с хуком для компоненты
    |   |    └── ...
    |   ├── Component.styles/    — Папка со стилями
    |   |    ├── index.ts        — Файл со стилями
    |   ├── Component.tests/     — Папка с unit-тестами для компоненты
    |   |    ├── index.ts        — Файл с тестами
    |   ├── Component.typings/   — Папка с интерфейсами для компонента
    |   |    ├── index.ts        — Файл с интерфейсами для компонента
    |   ├── Component.utils/     — Папка с хелперами для компоненты
    |   |    ├── firstUtil.ts    — Файл с утилитой для компоненты
    |   |    ├── secondUtil.ts   — Файл с утилитой для компоненты
    |   |    └── ...
    |   ├── Component.tsx        — Сам компонент
    |   ├── Component.md         — README файл компоненты
├── hooks/                       — Декларации бандлов
├── ui/                          — Переиспользуемые компоненты
|   ├── Component/               — Папка реакт компоненты
    |   |    └── ...             — Вся структура как для обычно компоненты
├── routes/                      — Все роуты приложения
├── pages/                       — Страницы приложения
|   ├── LoginPage.tsx            — Страница логина
|   ├── MainPage.tsx             — Главная страница
|   └── ...
├── store/                       — Папка со стором редакса
|   ├── index.ts                 — Файл стора
├── reducers/                    — Папка со слайсами
|   ├── firSlice/                — Папка слайса
    |   ├── firSlice.ts          — Файл слайса
    |   ├── firlice.typings.ts   — Интерфейсы для слайса
    |   └── ...    
|   ├── rootSlice                — Главный слайс
├── utils/                       — Глобальные утилиты
|   ├── firstUtil.ts             — Файл с утилитой
|   └── ...        
├── providers/                   — Провайдеры приложения
|   ├── ThemeProvider.tsx        — Файл с провайдером
|   └── ...        
├── docs/                        — Папка с документацией
|   ├── docs.md                  — Файл с документацией
|   └── ...        
├── styles/                      — Папка с глобальными css стилями
|   ├── style.css                — Файл со стилями
|   └── ...
```
